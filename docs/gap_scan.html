買い方ガイド — 具体的な手順

ざっくり言うと

前日より2%以上安く始まった株を朝イチで買い、元の値段に戻ったら売る。戻らなかったら3日以内に手仕舞い。

STEP 1 — 朝チェック（8:00）

この画面を開いて、下の表を見る。

  • ENTRY が付いている銘柄 → 買い候補。スコア55以上が目安
  • CAUTION → ニュースを確認してから判断
  • SKIP → 触らない

STEP 2 — ニュース確認（8:00-8:50）

銘柄コード（青リンク）をクリックしてYahoo Financeを開く。以下をチェック：

ニュース内容判断
特になし / 米国株安の影響だけ買ってOK（一時的な下げ）
決算発表で業績悪化買わない（本質的な下げ）
下方修正・減配買わない
アナリスト格下げ慎重に判断
不祥事・訴訟買わない

STEP 3 — 板情報チェック（8:55）

証券会社のアプリで、その銘柄の板（注文状況）を見る。

板の状態判断
買い注文が多い / 買い板が厚い反発しそう → 買う
売り注文だらけ / 気配値がどんどん下がるまだ下がる → 見送り

STEP 4 — 買う（9:00）

寄り付き（9:00）で成行注文を出す。同時に以下の2つの注文も入れる：

注文価格意味
利確（指値売り） 前日終値 窓埋め完了 = ゴール。
例: 前日3,000円で始値2,910円なら、3,000円で売り指値
損切り（逆指値売り） 買値の -1.5% これ以上下がったら自動で損切り。
例: 2,910円で買ったら、2,866円で逆指値

→ 注文を出したらあとは放置。利確か損切りが自動で約定する。

STEP 5 — 3日ルール

利確にも損切りにもかからず3日経ったら引けで売る。

窓埋めしない = この下げは戻らない可能性が高い。ズルズル持たない。

絶対やらないこと

NG行動理由
決算発表後のGDを買う業績悪化は「適正な下げ」。戻らない
損切りラインを外す・ずらす「もう少し待てば戻る」は危険。機械的に切る
全額一点集中連敗で退場する。資金の30-70%まで
SKIP銘柄に手を出す低ボラ・低フィル率は構造的に負ける

いくら投入するか（ポジションサイズ）

GDの大きさが、その銘柄の普段の値動き（ATR）の何倍かで決める。

GD幅 / ATR投入額考え方
1倍以下資金の 70%普段の範囲内。自信を持っていい
1〜2倍資金の 50%やや大きいが許容範囲
2〜3倍資金の 30%異常に大きい。控えめに
3倍以上資金の 30%（最小）何か起きている。最小ロット

例: 資金5万円、ATR 3%の銘柄が4%GD → GD/ATR = 1.3倍 → 5万円の50% = 25,000円分買う

具体例（全体の流れ）

資金5万円。INPEXがスコア77で ENTRY。

前日終値: 4,147円
GD想定価格: 4,064円（-2%）
ATR: 5.5%

8:00 スキャナーで確認。スコア77、ENTRY。ニュース = 特になし
8:55 板情報 → 買い板厚い。GO判断
9:00 始値 4,020円（GD -3.1%）で成行買い
　　 GD/ATR = 3.1/5.5 = 0.56倍 → 資金の70% = 35,000円 → 8株購入
　　 利確指値: 4,147円（前日終値）
　　 逆指値: 3,960円（買値-1.5%）

翌日 高値4,160円 → 4,147円で利確!
利益: (4,147 - 4,020) × 8株 = +1,016円（+3.2%）

戦略の仕組み — なぜギャップは埋まるのか

ギャップ発生のメカニズム

東証は板寄せ方式で始値を決定します。15:00の閉場から翌9:00の開場まで18時間の空白があり、その間に米国市場の動き・決算発表・為替変動などが起きます。翌朝、それらを織り込んだ注文が一括マッチングされるため、前日終値から大きく乖離した始値（ギャップ）が発生します。

なぜ「過剰反応」が起きるか

  • 個人投資家の感情的な売り — 夜間ニュースを見て怖くなり、成行売り注文を出す
  • 機関投資家は冷静 — 適正価格を計算して指値で買いを入れている
  • 流動性の断絶 — 18時間取引できないため、売り注文が溜まり始値に一気に反映される

→ この戦略は機関と同じ側に立って、個人の恐怖を拾う構造です。

窓が埋まる条件 / 埋まらない条件

埋まる（過剰反応）埋まらない（本質的な変化）
原因感情的な売り、需給の歪み決算ミス、業績悪化、構造変化
出来高少ない（個人の投げ売り）急増（機関が本気で売っている）
ATR対比普段の値動き範囲内普段の2-3倍の異常な動き
市場個別要因 or 全面パニック8-14銘柄のじわ売り
