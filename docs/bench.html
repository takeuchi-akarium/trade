Usage Guide
使い方を表示 ▶
3つのベンチタイプ
backtest実データで戦略のパフォーマンスを評価。リターン・MDD・Sharpe・月次リターン等を出力
scenario合成データ6パターン（下落・レンジ・V字回復・ダラダラ下落・バブル崩壊・レンジブレイク）で相場耐性を評価
allocation配分パターン比較。strategies=all → DART専用モード（現行ミックス/得意戦略のみ/段階制）、それ以外 → 等配分 vs 各戦略単独
Strategies の指定方法
all全登録戦略
short_termカテゴリ名で一括指定（short_term / long_term / composite / pair）
rsi,bb,ema戦略名をカンマ区切りで個別指定
dart,bb,ema_donDART含む場合はDART+サブ戦略を指定
パラメータ
Symbol銘柄（デフォルト: BTCUSDT）
Interval時間足（1d / 4h / 1h）
Years評価期間（年数）
SL %ストップロス。空欄=なし
TP %テイクプロフィット。空欄=なし
Tips: 結果は data/simulations/bench/ に自動保存。同条件は上書き。カードをクリックで詳細テーブル展開。
