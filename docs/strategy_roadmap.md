# 自動売買戦略ロードマップ

2026-04-11 時点。やりやすいものから順に実装する。

## 実装優先度

### 1. デュアルモメンタム (GEM)
- **概要**: 米国株 vs 非米国株の強い方を選び、Tビル(短期国債)より弱ければ債券に退避
- **頻度**: 月1回リバランス
- **資金目安**: 5万円~
- **実装難易度**: 低
- **備考**: batch_morning.py に統合可能。Gary Antonacci 提唱

### 2. ファンディングレート裁定 (BTC)
- **概要**: 現物ロング + 無期限先物ショートのデルタニュートラル。8h毎のファンディングレートを受取り
- **頻度**: 常時監視（ポジション管理・マージン管理）
- **資金目安**: 5万円~
- **実装難易度**: 中
- **備考**: ccxt で実装。年率10-20%。海外取引所(Binance/Bybit)前提

### 3. 高配当株スクリーニング
- **概要**: 配当利回り・ROE・自己資本比率・連続増配で機械的にスクリーニング
- **頻度**: 四半期リバランス
- **資金目安**: 100万円~
- **実装難易度**: 低
- **備考**: J-Quants API で財務データ取得

### 4. ペアトレード (日本株)
- **概要**: 共和分ペア(例: トヨタ vs ホンダ)のスプレッド平均回帰
- **頻度**: 日次監視
- **資金目安**: 30万円~（信用取引前提）
- **実装難易度**: 中
- **備考**: 立花証券 e支店 API で発注。statsmodels で共和分検定

### 5. イベントドリブン (TDnet 拡張)
- **概要**: 既存 TDnet バッチを拡張 → 業績修正・TOB で自動発注
- **頻度**: リアルタイム（開示検知時）
- **資金目安**: 10万円~
- **実装難易度**: 中~高
- **備考**: PEAD (Post-Earnings Announcement Drift) を狙う

### 6. FX キャリートレード
- **概要**: 円売り / 高金利通貨買いでスワップポイント収入
- **頻度**: 日次~週次
- **資金目安**: 5万円~
- **実装難易度**: 低~中
- **備考**: OANDA API (oandapyV20)。VIX 連動で退避判定
