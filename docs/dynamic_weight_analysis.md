## ファンダスコアによる補正 (2026-04-12)

テクニカル（SMA50）の判定タイミングをファンダで微調整する。

### ファンダスコア構成

3指標のZ-score合算（相関強度ウェイト）:
- ゴールド50日モメンタム（反転）: weight 0.41, 相関 -0.31
- 10年債20日変化: weight 0.37, 相関 +0.28
- FnG変化速度（7d-30d）: weight 0.22, 相関 +0.17

合成スコアとBTC 30日後リターンの相関: **+0.345**

### 補正ロジック

**Early Transition（タイミング調整）:**
- 上昇圏(dev 0~5%) + ファンダ弱気(<-0.3) → rangeに早期退避
- 下落圏(dev 0~-10%) + ファンダ強気(>0.3) → rangeに留まる（全退避しない）

**Boost（確信度ブースト）:**
- uptrend + ファンダ強気(>0.5) → ema_don 90% / bb 10%
- range + ファンダ強気(>0.5) → ema_don 20%に増量

### シナリオテスト結果

| | Tech only | Early+Boost |
|---|---|---|
| 期待リターン | +168.2% | **+180.7%** |
| 最悪ケース | +83.9% | **+93.1%** |
| 全シナリオ+ | Yes | Yes |

## 注意事項

- バックテストのリバランスコストは片道0.1%×2で計算。実際のスリッページは未考慮
- 50日MAの+1599%(±5%)等の極端な数値は過学習の可能性あり
- 下落時にbb_lsを有効化する場合、bb_ls戦略自体の改善が前提
- ファンダスコアの相関0.345は統計的に有意だが、比率を大きく動かすほど強くない
- ファンダはレジーム判定のタイミング調整として使うのが最適（直接weight変更は逆効果）
